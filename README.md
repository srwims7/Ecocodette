# Ecocodette
AI for Sustainability — Greener Route Web App

## Run backend
```bash
cd ecoCodette
mvn spring-boot:run
```

Backend APIs are available at:
- `GET /api/routes`
- `POST /api/routes`
- `GET /api/routes/summary`

## Run frontend
Open `frontend/index.html` in a browser after the backend starts.
